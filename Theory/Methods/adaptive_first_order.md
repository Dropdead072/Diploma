# Адаптивные методы первого порядка

##  AdaGrad (Adaptive Gradient)

$$
g_k = \nabla f(x_k) \\
H_k = [ \sum_{i=0}^k \text{diag}(g_i)^2 ]^{\frac{-1}{2}} \\
x_{k+1} = x_k - \eta H_k g_k
$$

Член $H_k$ называется оператором предобуславливания. 

Для большего удоства можно переписать альтернативно:

$$
g_k = \nabla f(x_k) \\
H_k = [ \sum_{i=0}^k \text{diag}(g_i)^2 ] \\
x_{k+1} = x_k - \eta H_k^{\frac{-1}{2}} g_k
$$


## RMSProp

Как можно заметить, у AdaGrad есть проблема, в предобуславливателе просто накапливаются квадраты градиентов с каждого шага, и если на каком-то шаге была ошибка, то вклад данного шага никак со временем не уменьшится. Для этого было предложен метод с некоторым аналогом импульса (momentum) для предобуславливателя.

$$
v_0 = 0 \\
v_{k+1} = v_k + (1 - \beta) \text{diag}(g_k)^2 \\
x_{k+1} = x_k - \eta v_{k+1}^{\frac{-1}{2}} g_k
$$
